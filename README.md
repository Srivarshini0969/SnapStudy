# SnapStudy

AI-Powered Lecture Revision & Timestamp Navigation Platform

## Live Demo

Frontend:
https://snap-study-six.vercel.app

## Features

- OCR Topic Detection
- YouTube Lecture Suggestions
- Timestamp Resume Navigation
- JWT Authentication
- Cloudinary Image Storage
- Smart Revision Tracking
- Dark Mode
- Responsive UI

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- Cloudinary

OCR:
- Tesseract.js

Deployment:
- Railway
- Vercel
